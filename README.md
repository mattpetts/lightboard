## 🚀 Overview

A cool little exercise in dom manipulation and OOP Javascript creating a customisable class that can render different size grids to the dom and create the lightboard effect.  

---

## ✨ Features

- ✅ Customisable
- 🚧 Drawing mode. Allow users to click on a tile to light it up and keep it in the lit up state allowing users to create pixel art
- 🚧 customisable colour pallette
- 🚧 make mobile responsive
